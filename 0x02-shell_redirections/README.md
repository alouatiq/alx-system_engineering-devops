## Project: Shell, I/O Redirections and filters

### Description
This project is part of the ALX System Engineering & DevOps program. It covers the following topics:

- Shell, I/O Redirection
- What do the commands `head`, `tail`, `find`, `wc`, `sort`, `uniq`, `grep`, `tr` do
- How to redirect standard output to a file
- How to get standard input from a file instead of the keyboard
- How to send the output from one program to the input of another program
- How to combine commands and filters with redirections
- Special Characters
- What are special characters
- Understand what do the white spaces, single quotes, double quotes, backslash, comment, pipe, command separator, tilde and how and when to use them
- Other Man Pages
- How to display a line of text
- How to concatenate files and print on the standard output
- How to reverse a string
- How to remove sections from each line of files
- What is the `/etc/passwd` file and what is its format
- What is the `/etc/shadow` file and what is its format

### Tasks
The project includes the following tasks:

1. `0-hello_world`: **Hello, World** - Write a script that prints "Hello, World", followed by a new line to the standard output.
2. `1-confused_smiley`: **Confused smiley** - Write a script that displays a confused smiley `"(Ã”o)'`.
3. `2-hellofile`: **Let's display a file** - Display the content of the `/etc/passwd` file.
4. `3-twofiles`: **What about 2?** - Display the content of `/etc/passwd` and `/etc/hosts`.
5. `4-lastlines`: **Last lines of a file** - Display the last 10 lines of `/etc/passwd`.
6. `5-firstlines`: **I'd prefer the first ones actually** - Display the first 10 lines of `/etc/passwd`.
7. `6-third_line`: **Line #2** - Write a script that displays the third line of the file `iacta`.
8. `7-file`: **It is a good file that cuts iron without making a noise** - Write a shell script that creates a file named exactly `\*\\'"Best School"\'\\*$\?\*\*\*\*\*:)` containing the text `Best School` ending by a new line.
9. `8-cwd_state`: **Save current state of directory** - Write a script that writes into the file `ls_cwd_content` the result of the command `ls -la`.
10. `9-duplicate_last_line`: **Duplicate last line** - Write a script that duplicates the last line of the file `iacta`.
11. `10-no_more_js`: **No more javascript** - Write a script that deletes all the regular files (not the directories) with a `.js` extension that are present in the current directory and all its subfolders.
12. `11-directories`: **Don't just count your directories, make your directories count** - Write a script that counts the number of directories and sub-directories in the current directory.
13. `12-newest_files`: **What's new** - Create a script that displays the 10 newest files in the current directory.
14. `13-unique`: **It must be in that file** - Display lines containing the pattern "root" from the file `/etc/passwd`.
15. `14-findthatword`: **Count that word** - Display the number of lines that contain the pattern "bin" in the file `/etc/passwd`.
16. `15-whatsnext`: **What's next?** - Display lines containing the pattern "root" and 3 lines after them in the file `/etc/passwd`.
17. `16-hidethisword`: **I hate bins** - Display all the lines in the file `/etc/passwd` that do not contain the pattern "bin".
18. `17-letteronly`: **Letters only please** - Display all lines of the file `/etc/ssh/sshd_config` starting with a letter.
19. `18-AZ`: **A to Z** - Replace all characters `A` and `c` from input to `Z` and `e` respectively.
20. `19-hiago`: **Without C, you would live in hiago** - Create a script that removes all letters `c` and `C` from input.
21. `20-reverse`: **esreveR** - Write a script that reverse its input.
22. `21-users_and_homes`: **DJ Cut Killer** - Write a script that displays all users and their home directories, sorted by users.
23. `100-empty_casks`: **Empty casks make the most noise** - Write a command that finds all empty files and directories in the current directory and all sub-directories.
24. `101-gifs`: **A gif is worth ten thousand words** - Write a script that lists all the files with a `.gif` extension in the current directory and all its sub-directories.
25. `102-acrostic`: **Acrostic** - Create a script that decodes acrostics that use the first letter of each line.
26. `103-the_biggest_fan`: **The biggest fan** - Write a script that parses web servers logs in TSV format as input and displays the 11 hosts or IP addresses which did the most requests.
